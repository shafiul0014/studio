# **App Name**: FlightConnect

## Core Features:

- Landing Page: Single-page landing page with clear call-to-action for flight booking, changes, and cancellations.
- Fixed Call Button: A prominently displayed, fixed 'Call Now' button that is always visible to the user with the number (844) 578-3079.
- Legal Pages: Inclusion of 'Privacy' and 'Terms' legal pages with relevant content.
- Responsive Design: Ensure the website is fully responsive and adapts to different screen sizes (desktop, tablet, mobile).

## Style Guidelines:

- Primary color: A calming blue (#5DADE2) to evoke trust and reliability, aligning with travel-related themes.
- Background color: A light gray (#F5F5F5), almost white, provides a clean backdrop that keeps the focus on the content.
- Accent color: A vibrant orange (#FFB347) is used for the 'Call Now' button and other interactive elements, creating contrast.
- Font: 'PT Sans' for a modern, yet slightly warm feel, suitable for headlines and body text.
- Simple, intuitive layout that immediately presents the service offerings and emphasizes the phone number.
- Use simple, clean icons to visually represent services such as new bookings, changes, and cancellations.
- Subtle animations (e.g., a gentle pulse on the 'Call Now' button) to draw attention without being distracting.
